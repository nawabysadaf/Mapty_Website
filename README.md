Welcome to **Mapty Website**, a practice web application developed as part of my learning journey in **advanced JavaScript, HTML, SASS, and API integration.** This project was created to reinforce the concepts I learned through an online JavaScript course, specifically focusing on working with real-world APIs such as the Google Maps API.
